# 🚀 Registration Management System

A full-stack user registration and authentication application built with **Python (Flask)** and **MySQL**. The application allows users to register, log in, manage their profiles, and securely stores user information in a MySQL database.

---

## 📌 Features

- ✅ User Registration
- ✅ User Login & Logout
- ✅ Secure Password Hashing
- ✅ Session Management
- ✅ Form Validation
- ✅ User Dashboard
- ✅ Edit Profile
- ✅ Delete Account
- ✅ MySQL Database Integration
  ## 🔄 Application Workflow

```
User
   │
   ▼
Registration Form
   │
   ▼
Flask Backend
   │
   ▼
Input Validation
   │
   ▼
MySQL Database
   │
   ▼
Login Authentication
   │
   ▼
Dashboard
```
